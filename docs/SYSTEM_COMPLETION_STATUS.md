## 不建议继续新增的大功能

暂时不建议继续新增：

- 多个新商品领域
- 复杂向量数据库部署
- 多角色大模型互评
- 复杂权限系统
- 在线支付/真实订单改写
- 复杂后台管理系统

这些会扩大工程范围，影响最终交付。
